## Competitor Landscape

| Competitor | What They Have | Gap Steps Can Own |
|---|---|---|
| **Strava** | Pace calc, buried in app | No water, HR zones, activity→steps; not SEO-optimized |
| **MyFitnessPal** | Macro, BMR (mostly paywalled) | No HR zones, VO2 max, race predictor |
| **Garmin** | VO2, training zones (device-locked) | Requires expensive hardware — Steps is free, web-based |
| **Calculator.net** | 100+ tools, dominant SEO | Zero shareability, dated design, no personalization |
| **Omni Calculator** | Beautiful tools, high authority | No app integration, no step-tracking angle |

**Steps App Advantage:** Free + web-based + mobile-first + integrates with step data + shareable results.

---

## Market Gaps (Steps App Can Own)

1. **Running metrics for casual runners** — Strava/Garmin assume serious athletes; Steps targets beginners
2. **No wearables required** — Garmin locks metrics to hardware; Steps offers age-based formula tools
3. **Weight loss calculator suite** — cohesive flow from steps → calories → TDEE → macros → deficit
4. **Activity-to-steps conversion** — no competitor does this; unique to step challenge use case
5. **Shareable results** — none of the top competitors have social share hooks on calculators

## Implementation Notes (Technical)

All new tools can follow the existing pattern:
```
/app/tools/[tool-name]/page.tsx     ← SEO metadata + server component
/app/tools/[tool-name]/client.tsx   ← Calculator UI (client component)
/lib/[tool-name]-calculator.ts      ← Calculation logic
```

**Reuse opportunities:**
- Water Intake + Hydration → extend `calorie-calculator.ts`
- Body Fat → standalone `body-fat-calculator.ts` (Navy method)
- Heart Rate Zones → standalone `heart-rate-calculator.ts`
- Activity→Steps → extend `step-calculator.ts` (MET values lookup)
- Pace Converter → standalone `pace-calculator.ts`

**Data flow synergies (future UX enhancement):**
- Weight entered in BMI → auto-fill TDEE/Macro/Calorie Deficit
- Step goal set → suggest Heart Rate Zones for optimal exercise
- Running pace entered → show equivalent steps + calories
