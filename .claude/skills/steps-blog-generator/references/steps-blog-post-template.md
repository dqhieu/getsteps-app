## Word Count Targets

| Section | Words |
|---------|-------|
| Intro | 100-150 |
| What is | 150-200 |
| How to | 300-400 |
| Tips | 200-300 |
| FAQ | 200-300 |
| Why it matters | 100-150 |
| CTA | 50-100 |
| **Total** | **1100-1600** |

## Keyword Placement

1. **Title** - Primary keyword near start
2. **Meta description** - Primary keyword + benefit
3. **H1** - Exact match or close variant
4. **First paragraph** - Within first 100 words
5. **H2 headings** - 2-3 headings include keyword
6. **Body** - Natural usage, 2-3% density
7. **Image filename** - Include keyword slug
8. **URL slug** - Primary keyword

## Internal Links

Always include:
- App Store download link in CTA
- Link to related blog posts (when available)

## Tone Guidelines

- **Voice**: Friendly, helpful, expert
- **Perspective**: Second person ("you")
- **Style**: Clear, concise, actionable
- **Avoid**: Jargon, fluff, over-promising
